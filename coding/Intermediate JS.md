# Intermediate JS

## Random number generator

```js
// n is randomly assigned values from 0 - 0.99999
// not including 1
var n = Math.random()

// scale up to 6
// generate random no from 0 to 5.9999
n *= 6

// round to the the lower value`
n = Math.floor(n)
```
