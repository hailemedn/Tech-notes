# Destructuring

```js
const rgb = [12, 165, 79];
```

inorder to grab the red, we use `rgb[0]`  
but this can get hard to read and tricky if we got a large lines of code.

we can use destructuring to map each item to a variable.

```js
const [red, green, blue] = [12, 165, 79];
```
